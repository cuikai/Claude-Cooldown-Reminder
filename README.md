# Claude Cooldown Reminder 🕒

A lightweight, open-source Chrome extension designed for Claude power users. It tracks your 5-hour rate limit cooldown and sends a native browser notification the exact moment your quota resets, ensuring you can seamlessly jump back into your workflow.

---

## ✨ Features

- **Token & Capacity Check:** Easily monitor your usage and track when your next limit window opens.
- **Smart Cooldown Tracking:** Automatically calculates the remaining time for Claude's 5-hour rate limit.
- **Native Desktop Notifications:** Sends a standard Chrome push notification when your cooldown ends, so you don't have to keep refreshing or guessing.
- **100% Free & Open Source:** Transparent codebase, zero tracking, completely focused on data privacy.

## 📦 Installation

### Method 1: Install from Chrome Web Store (Recommended)
You can install the extension directly from the official store:
👉 [Claude Cooldown Reminder - Chrome Web Store](https://chromewebstore.google.com/detail/claude-cooldown-reminder/nfbeppceneplfkcengfibhbnddcdjfie?hl=en)

### Method 2: Manual Installation (For Developers)
1. Clone this repository: `git clone https://github.com/cuikai/Claude-Cooldown-Reminder.git`
2. Open your Chrome browser and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (top right toggle).
4. Click **Load unpacked** (top left button) and select the cloned repository folder.

## 🔒 Privacy & Security

This extension runs entirely locally in your browser. 
- **No external servers:** All tracking and countdown calculations are processed locally.
- **No data collection:** Your interaction data with Claude is never saved, uploaded, or shared.
